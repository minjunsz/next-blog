    Likelihood Maximize 관점에서 Loss 정의

    1. 우리의 목표는 input X에 대해 결과 Y가 뭐가 될 것인가? 라는 질문에 답하는 것이므로 conditional probability P(Y∥X)P(Y\|X)P(Y∥X)를 찾고자 한다.
    2. Conditional probability를 근사하기 위해 parametrized distribution P(Y∥paramter w)P(Y\|paramter\ w)P(Y∥paramter w)를 우리가 원하는 함수 형태로 정의한다.
    3. ML model의 output은 확률 분포에 사용될 parameter라고 생각한다. 그 결과 생성된 분포는 P(Y∥w=fθ(X))P(Y\|w=f_\theta(X))P(Y∥w=fθ​(X)) 이 되고, 이것을 0번에서 언급한 conditional probability라고 생각한다.
    4. samplei=(xi,yi)sample_i=(x_i, y_i)samplei​=(xi​,yi​)에 대해서 생각해보면 input이 xix_ixi​일 때 y값에 대한 확률 분포는 P(Y∥fθ(xi))P(Y\|f_\theta(x_i))P(Y∥fθ​(xi​))가 되고 실제로 sample i와 같은 결과를 얻을 확률은 P(Y=yi∥fθ(xi))P(Y=y_i\|f_\theta(x_i))P(Y=yi​∥fθ​(xi​)) 가 된다. Notation의 편의를 위해 P(Y=yi∥fθ(xi))=p(yi∥fθ(xi))P(Y=y_i\|f_\theta(x_i)) = p(y_i\|f_\theta(x_i))P(Y=yi​∥fθ​(xi​))=p(yi​∥fθ​(xi​)) 라고 쓴다. 알고싶은 변수 θ\thetaθ가 주어졌을 때 관측값 yiy_iyi​에 대한 확률이므로 likelihood라고 볼 수 있다.
    5. 개별 sample에 대한 likelihood가 결정됐다면 전체 sample에 대한 확률도 생각해볼 수 있다. i.i.d. 조건 하에서 전체 likelihood는 개별 likelihood의 곱으로 표현된다. p(Y∣θ,X)=∏ip(yi∣fθ(xi))p(Y|\theta, X) = \prod_i p(y_i|f_\theta(x_i))p(Y∣θ,X)=∏i​p(yi​∣fθ​(xi​))
    6. 우리의 목표는 이 likelihood를 maximize하는 건데 loss function은 곱이 아니라 합으로 표현돼야 하고 minimize하는 형태로 정의되야하므로 negative log likelihood(NLL)를 minimize한다.
    max⁡θ∏ip(yi∣fθ(xi))  ⟺  max⁡θ∑ilog⁡p(yi∣fθ(xi))  ⟺  min⁡θ∑i−log⁡p(yi∣fθ(xi))\begin{align*} \max_\theta \prod_i p(y_i|f_\theta(x_i)) &\iff \max_\theta \sum_i \log p(y_i|f_\theta(x_i)) \\ &\iff \min_\theta \sum_i -\log p(y_i|f_\theta(x_i)) \end{align*}θmax​i∏​p(yi​∣fθ​(xi​))​⟺θmax​i∑​logp(yi​∣fθ​(xi​))⟺θmin​i∑​−logp(yi​∣fθ​(xi​))​

    그럼 1번에서 정의한 Noise distribution이 어떤 형태인지에 따라서 loss function이 다르게 된다.

    • Gaussian Distribution을 선택한 경우 -> Mean Squared Error가 된다. Link 참조
    • Multinomial Distribution을 선택한 경우 -> Cross Entropy가 된다. Link 참조
